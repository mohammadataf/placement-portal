# 🚀 Placement Portal (MERN Stack)

A full-stack Placement Portal built using the MERN stack. This project is being developed feature-by-feature following industry-standard architecture and Git practices.

## 📌 Tech Stack

### Frontend

* React
* Vite
* JavaScript

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* Dotenv

### Tools

* Git & GitHub
* Nodemon
